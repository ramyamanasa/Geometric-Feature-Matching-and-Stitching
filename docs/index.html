Pipeline

Six tightly coupled stages take a raw image pair to a seamless panorama.

1
Structure Tensor

Harris Corner Detection

The Harris detector analyses the second-moment (structure) tensor of image gradients. A corner is a location where both tensor eigenvalues are large — meaning gradient energy varies strongly in all directions. Rather than computing eigenvalues explicitly, the response det(M) − k·tr(M)² is used as an efficient approximation. Corners within a 20-pixel border strip are discarded to avoid unreliable boundary effects.

2
Local Max Filtering

Non-Maximal Suppression (NMS)

Harris responses cluster densely around strong edges and corners. A sliding 11×11-pixel window scans each detection: only the single pixel with the highest response in its neighbourhood survives. This reduces ~26,000 raw detections to ~4,900 well-distributed keypoints per image without discarding strong corners.

3
MOPS-Style · 192-dim

RGB Patch Descriptor Extraction

A 40×40 RGB patch is sampled around each surviving keypoint and downsampled to 8×8×3 using anti-aliased bicubic interpolation, producing a 192-dimensional vector. The descriptor is bias/gain normalised ((d − μ) / σ) to achieve invariance to affine illumination changes. Using RGB rather than grayscale preserves colour discriminability — a red and blue object with identical texture are clearly distinguishable.

4
Lowe's Ratio Test

NNDR Feature Matching

For each descriptor in image 1, the two nearest neighbours in image 2 are found using squared-Euclidean (SSD) distance. A match is accepted only when d(1NN) / d(2NN) < 0.7 — meaning the closest match is significantly more distinctive than the runner-up. This ratio test eliminates the majority of ambiguous correspondences in repetitive or textureless regions.

5
DLT + SVD · RANSAC

Robust Homography Estimation

RANSAC iteratively draws random 4-match subsets and fits a planar homography via the Direct Linear Transform (DLT): each correspondence contributes two rows to a linear system Ah = 0, solved by extracting the last right-singular vector via SVD. The hypothesis maximising geometrically consistent inliers (reprojection error < 5 px) is retained. With 2000 iterations, the algorithm reliably identifies the correct transformation even when 10–30 % of input matches are outliers.

6
Bilinear Interpolation · Average Blend

Inverse Warping & Compositing

Every output pixel is back-projected through H⁻¹ to its source location in image 1. Sub-pixel colours are resolved by bilinear interpolation over the four nearest source pixels, avoiding the holes that forward-warping produces. The canvas is sized to contain both transformed images; overlapping regions are composited by averaging pixel values.
